# Customer Behavior Data Analyst Portfolio Project
Customer Shopping Behavior Analysis is a data analytics project that explores 3,900 retail transactions to uncover insights into customer spending, product preferences, and subscription trends. Using Python, SQL, and Power BI, the project delivers business insights through data cleaning, analysis, and interactive dashboards.
# Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior using transactional data to uncover insights into spending patterns, product preferences, customer segments, and subscription trends. The project demonstrates an end-to-end data analytics workflow, from data cleaning and exploration to business intelligence reporting and dashboard creation.

## Dataset

The dataset contains customer demographics, purchase details, shopping behavior, and transaction-related information.

**Key Features:**

* Age, Gender, Location
* Item Purchased, Category
* Purchase Amount
* Subscription Status
* Review Rating
* Discount Applied
* Shipping Type
* Purchase Frequency

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib)
* **BigQuery SQL**
* **Power BI**
* **Gamma** (Presentation Design)
* **GitHub**

## Project Workflow

### 1. Data Loading & Exploration

* Imported and inspected the dataset using Python.
* Performed Exploratory Data Analysis (EDA) to understand distributions, trends, and relationships.

### 2. Data Cleaning

* Handled missing values.
* Standardized column names.
* Created additional features for analysis.
* Validated data consistency.

### 3. SQL Analysis (BigQuery)

* Revenue analysis by customer segments.
* Product performance evaluation.
* Subscription vs non-subscription comparison.
* Customer segmentation and behavioral analysis.

### 4. Dashboard Development

* Built an interactive Power BI dashboard.
* Visualized KPIs, revenue trends, customer behavior, and product insights.

### 5. Reporting & Presentation

* Created a detailed project report summarizing findings and recommendations.
* Designed a presentation in Gamma to communicate insights effectively.

## Key Results

* Identified top-performing products and categories.
* Analyzed spending behavior across customer groups.
* Evaluated the impact of subscriptions and discounts on revenue.
* Generated actionable business recommendations.

## How to Run

1. Clone this repository.
2. Install required Python libraries.
3. Run the Jupyter Notebook or Python script for data preprocessing and EDA.
4. Execute SQL queries in BigQuery.
5. Open the Power BI dashboard file (.pbix) to explore visualizations.
6. Review the report and presentation for detailed findings.
